\# Forms



\## Purpose



Defines standards for data entry.



\---



\## Principles



Forms should:



\- Minimize typing

\- Reduce user errors

\- Support keyboard navigation



\---



\## Validation



Validation should occur:



\- During input when appropriate.

\- Before submission.



\---



\## Default Values



Provide sensible defaults whenever possible.



\---



\## Error Display



Errors should:



\- Clearly identify the field.

\- Explain how to resolve the issue.



\---



\## Principles



\- Efficiency

\- Clarity

\- Error prevention
