\# Navigation



\## Purpose



Defines navigation standards.



\---



\## Navigation Principles



Users should always know:



\- Current location

\- Available actions

\- Navigation path



\---



\## Navigation Types



\- Main Navigation

\- Context Navigation

\- Dialog Navigation



\---



\## Back Navigation



Back navigation should preserve user context whenever practical.



\---



\## Dialogs



Dialogs should:



\- Focus on a single task.

\- Require explicit confirmation when necessary.



\---



\## Principles



\- Predictability

\- Minimal navigation depth
