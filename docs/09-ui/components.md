\# Components



\## Purpose



Defines reusable UI components.



\---



\## Standard Components



\- Buttons

\- Inputs

\- Selects

\- Tables

\- Cards

\- Lists

\- Dialogs

\- Notifications



\---



\## Component Principles



Components should be:



\- Reusable

\- Consistent

\- Accessible

\- Predictable



\---



\## Interaction



Each component should provide immediate visual feedback.



\---



\## States



Components may support:



\- Default

\- Hover

\- Focus

\- Active

\- Disabled

\- Loading
