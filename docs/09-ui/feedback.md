\# User Feedback



\## Purpose



Defines user feedback standards.



\---



\## Feedback Types



\- Loading

\- Success

\- Information

\- Warning

\- Error

\- Confirmation



\---



\## Loading



Users should always know when processing is in progress.



\---



\## Success



Successful operations should provide confirmation.



\---



\## Errors



Errors should explain:



\- What happened.

\- What the user can do next.



\---



\## Confirmation



Require confirmation only for irreversible operations.



\---



\## Principles



\- Immediate

\- Clear

\- Actionable
