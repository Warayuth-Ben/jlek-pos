\# Accessibility



\## Purpose



Defines accessibility requirements.



\---



\## Keyboard Support



All major functions should be accessible using the keyboard.



\---



\## Focus



Visible focus indicators must always be provided.



\---



\## Contrast



Text and interactive elements should maintain sufficient contrast.



\---



\## Touch Targets



Interactive elements should provide adequate touch target sizes.



\---



\## Accessibility Principles



\- Keyboard accessible

\- Screen reader friendly

\- Sufficient contrast

\- Consistent interaction
