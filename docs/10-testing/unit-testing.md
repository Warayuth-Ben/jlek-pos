\# Unit Testing



\## Purpose



Defines standards for unit testing.



\---



\## Scope



Unit Tests verify individual units of behavior.



\---



\## Targets



\- Domain Services

\- Value Objects

\- Aggregates

\- Utility Functions



\---



\## Principles



\- Independent

\- Repeatable

\- Fast

\- Deterministic



\---



\## Assertions



Tests should verify:



\- Expected results

\- Invalid inputs

\- Edge cases
