\# JLek POS Knowledge Book



> เรื่องเล่าของร้านอาหาร ที่ซ่อน Software Architecture ไว้ในทุกเหตุการณ์



\---



\## Purpose



JLek POS Knowledge Book เป็นหนังสือที่รวบรวมองค์ความรู้ของร้านอาหาร โดยใช้ JLek POS เป็นกรณีศึกษา



หนังสือเล่มนี้ไม่ได้มีเป้าหมายเพื่ออธิบายการใช้งานระบบ แต่มีเป้าหมายเพื่อถ่ายทอดวิธีคิด เหตุผล และประสบการณ์ที่อยู่เบื้องหลังการออกแบบระบบ



\---



\## Objectives



\- ถ่ายทอด Workflow ของร้านอาหาร

\- อธิบายเหตุผลของ Business Rules

\- เชื่อม Business กับ Software Architecture

\- บันทึกองค์ความรู้ของร้าน

\- ส่งต่อความรู้ให้คนรุ่นต่อไป



\---



\## Audience



หนังสือเล่มนี้เหมาะสำหรับ



\- เจ้าของร้าน

\- ผู้จัดการร้าน

\- Developer

\- QA

\- System Analyst

\- นักศึกษา

\- ผู้ที่สนใจ Domain-Driven Design
