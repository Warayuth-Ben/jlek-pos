\# Book Vision



\## Vision



หนังสือเล่มนี้ไม่ได้ถูกสร้างขึ้นเพื่ออธิบายโปรแกรม



แต่ถูกสร้างขึ้นเพื่อบันทึกองค์ความรู้ของร้าน



เป้าหมายคือให้คนรุ่นต่อไปเข้าใจ



\- ร้านทำงานอย่างไร

\- ระบบทำงานอย่างไร

\- และเหตุใดระบบจึงถูกออกแบบเช่นนั้น



\---



\## Mission



เราไม่ได้เขียนหนังสือเพื่ออธิบาย Source Code



เราเขียนหนังสือเพื่อบันทึกวิธีคิด



เพื่อให้ความรู้ของร้านไม่สูญหายไปตามกาลเวลา



\---



\## Motto



อย่าบันทึกเพียงว่า



"ระบบทำอะไร"



แต่จงบันทึกว่า



"ร้านคิดอย่างไร"



\---



\## Long-term Goal



หนังสือเล่มนี้ควรอ่านได้แม้อีก 10 ปีข้างหน้า



แม้เทคโนโลยีจะเปลี่ยน



แต่เหตุผลเบื้องหลังการออกแบบระบบควรยังคงอยู่



\---



\## Success Criteria



หากวันหนึ่ง



คนที่ไม่เคยรู้จัก JLek POS



อ่านหนังสือเล่มนี้



แล้วเข้าใจว่า



"ทำไมร้านจึงทำงานเช่นนี้"
