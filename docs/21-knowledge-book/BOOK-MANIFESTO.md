\# Book Manifesto



หนังสือเล่มนี้ไม่ได้เกิดขึ้น

แต่มันเกิดขึ้น

เพราะเราไม่อยากให้ความรู้ของร้านหายไป



ซอฟต์แวร์สามารถเขียนใหม่ได้



Framework สามารถเปลี่ยนได้



Database สามารถย้ายได้



แต่เหตุผลของการตัดสินใจ

ไม่ควรสูญหาย



ทุกระบบมี Source Code



แต่ไม่ใช่ทุกระบบจะมีเรื่องราว



JLek POS จะมีทั้งสองอย่าง
