\# Security Guidelines



\## Purpose



Define minimum security requirements.



\---



\## Authentication



All authenticated users must be verified.



\---



\## Authorization



Users may only perform permitted actions.



\---



\## Input Validation



Validate all external input.



\---



\## Sensitive Data



Never expose:



\- Passwords

\- Tokens

\- Secrets



\---



\## Logging



Never log:



\- Passwords

\- Authentication tokens



\---



\## Updates



Keep dependencies updated.
