\# Payment Domain Overview



\## Purpose



Defines the domain responsible for financial transactions within the restaurant POS system.



The Payment domain manages bill calculation, payment processing, refunds, receipt generation, and bill closure.



\---



\## Responsibilities



The Payment domain is responsible for:



\* Creating bills

\* Calculating payable amounts

\* Processing payments

\* Recording refunds

\* Closing bills

\* Publishing payment events



\---



\## Collaborates With



\* Ordering Domain

\* Table Domain



The Payment domain never modifies customer orders directly.
