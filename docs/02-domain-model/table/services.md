\# Table Domain Services



\## Purpose



Defines business operations within the Table domain.



\---



\# Table Assignment Service



Assigns an Order Session to a table.



\---



\# Table Transfer Service



Moves an active Order Session to another table.



\---



\# Table Merge Service



Combines multiple tables into one dining session.



\---



\# Table Split Service



Separates a merged dining session.



\---



\## Principles



\* Services operate on Dining Table aggregates.

\* Services preserve business history.
