\# Table Domain Overview



\## Purpose



Defines the domain responsible for managing restaurant tables and dining sessions.



The Table domain manages seating assignments, table status, transfers, merges, and splits.



\---



\## Responsibilities



The Table domain is responsible for:



\* Managing table status

\* Assigning tables

\* Transferring tables

\* Merging tables

\* Splitting tables



\---



\## Collaborates With



\* Ordering Domain

\* Payment Domain



The Table domain does not own customer orders or payments.
