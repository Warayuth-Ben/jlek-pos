\# Shared Domain Overview



\## Purpose



Defines domain concepts shared across multiple bounded contexts.



These concepts are independent of a specific business process and provide a common language throughout the system.



Shared domain objects must remain generic and reusable.



\---



\## Scope



This section includes:



\* Value Objects

\* Shared Domain Events

\* Common Domain Services



Business entities that belong to a specific context must not be defined here.
