\# Glossary



| Term | Definition |

|------|------------|

| Aggregate | Cluster of Domain Objects treated as a consistency boundary |

| Entity | Object with identity |

| Value Object | Immutable object defined by value |

| Domain Event | Business event that already occurred |

| Application Service | Coordinates use cases |

| Repository | Persistence abstraction |

| State Machine | Defines lifecycle transitions |

| Business Rule | Business constraint |

| Use Case | Application interaction scenario |

| Kitchen Ticket | Kitchen production document |

| Add-on Order | Additional order after confirmation |

| Bill | Payment document |

| Payment | Financial transaction |
