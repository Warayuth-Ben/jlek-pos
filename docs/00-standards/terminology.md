\# Terminology



\## Order



A customer's food request before payment.



\---



\## Bill



A financial record used for payment.



\---



\## Kitchen Ticket



A production instruction sent to the kitchen.



\---



\## Add-on Order



A new order created after the original order has already been confirmed.



\---



\## Table



A dine-in seating location.



\---



\## Takeaway



An order without table assignment.
